Enterprise Network Infrastructure & Remote Management

A Comprehensive IT Systems Project for Secure and Manageable Network Services

Project Overview

As a core component of my Cicle de Grau Mitjà de Informàtica, this project involved the end-to-end design, implementation, and management of a secure network infrastructure for a simulated small business environment. The goal was to build a robust and scalable network from the ground up, integrating industry-standard hardware and open-source solutions.

The system was built around a pfSense firewall for maximum security and control. Core network services including DNS, FTP, and a web server were deployed on a Windows Server. The project culminated in a successful demonstration of secure remote administration using TeamViewer, proving the reliability and accessibility of the entire infrastructure.

Technologies & Skills

pfSense
pfSense Firewall: Deployed and configured an open-source pfSense firewall to manage all network traffic, establish security rules, handle NAT, and act as the primary gateway router.
Cisco IOS
Cisco IOS: Configured Cisco routers and switches via the command-line interface (CLI) for efficient data packet management, inter-VLAN routing, and core network infrastructure setup.
Windows Server
Windows Server: Installed and administered a Windows Server instance to host critical network services required for the business environment.
DNS Management
DNS Management: Set up and managed a DNS server to handle local name resolution, allowing devices to communicate using hostnames instead of IP addresses.
FTP Server
FTP Server: Implemented a File Transfer Protocol (FTP) server, enabling secure and efficient file sharing across the network.
HTML / Web Server
HTML / Web Server: Deployed a basic web server to host an internal company website, demonstrating the ability to provide web-based services on the network.
Remote Administration
Remote Administration (TeamViewer): Configured secure remote access to the server, allowing for system administration and troubleshooting from any location, as demonstrated in the final presentation.
Network Troubleshooting
Network Troubleshooting: Demonstrated practical problem-solving skills by diagnosing and resolving issues with server configurations and network connectivity to ensure project success.

Key Challenges & Process

Network Design: The first phase involved designing the network topology, defining IP address schemes, and planning VLANs to segment traffic for security and performance. This foundational step ensured a logical and scalable network structure.

Implementation & Configuration: The core of the project was the hands-on configuration of the hardware and software. This included installing pfSense from scratch, defining firewall rules to allow legitimate traffic while blocking threats, and configuring Cisco devices for optimal data flow. On the server side, roles for DNS, web (IIS/Apache), and FTP services were installed and configured to be accessible to network clients.

Problem-Solving in Action: The project presented real-world challenges, including troubleshooting the Windows Server instance to ensure stable operation. Overcoming these hurdles required a systematic approach to diagnostics and demonstrated resilience and a practical understanding of system administration.

Validation: The final proof of concept was a live demonstration showing a remote connection from a separate laptop to the server via TeamViewer. This single action successfully validated that the firewall rules, network routing, server services, and remote access software were all configured correctly and working in harmony.

Design Rationale

This project is a testament to the practical application of fundamental networking and systems administration principles. It proves an ability to not only understand the theory behind secure networks but to actually build one using a combination of proprietary (Cisco) and open-source (pfSense) technologies. The focus on security, remote manageability, and the deployment of essential business services makes this a direct reflection of the skills required in a modern IT environment.
