PARAMETRIC MOUSE

Fully customisable mouse created with Rhino Grasshopper

Type: Individual Project  |  Time: 1 day  |  Team: Individual

Project Overview

This project explores how far parametric design can be pushed within the constraints of a familiar everyday object: the computer mouse. Using Rhino Grasshopper, the entire mouse geometry is defined procedurally, allowing the designer to generate radically different forms, surfaces, and scroll-wheel patterns from a single script.

The core idea is simple: instead of sculpting one “final” shape, the mouse becomes a configurable system. Key dimensions and profiles can be tuned down to 1 mm precision, enabling ergonomic fine‑tuning, rapid form exploration, and highly personalised design outcomes.

The result is a design tool that can be used to create a whole family of mice—from subtle ergonomic tweaks to extreme concept forms—without rebuilding the model from scratch.

Shape Formation & Body Modelling

The mouse body is generated from a series of control points that can be moved along the X, Y, and Z axes. By adjusting these points and their associated curves, the designer can freely mould almost any mouse silhouette while maintaining a clean, continuous surface.

A graph mapper drives the section profiles along the length of the mouse, allowing the height, width, and curvature of both the front and rear to be tuned in real time. This makes it possible to quickly move from a low, aggressive gaming profile to a taller, more supportive ergonomic shape, without rebuilding the geometry.

Scroll Wheel & Coded Disc

The parametric system extends beyond the outer shell to the scroll wheel and internal coded disc. By drawing a simple 2D curve, any ridge pattern for the scroll wheel can be defined and then projected and cut into the wheel surface.

Key characteristics—such as ridge size, radius, spacing, and connector length—are all controlled by parameters. The same logic is used to generate a coded disc: a designer can sketch almost any profile, which is then processed by Grasshopper to create a disc with matching cut‑outs, ready to interface with an encoder or sensor system.

Design Variations

The final definition is capable of producing a wide range of mouse designs from a single script. Small adjustments in the graph mapper or point positions yield noticeably different bodies, while more extreme changes produce entirely new concepts that still remain manufacturable.

These variations demonstrate how parametric workflows can turn a one‑off model into a flexible design platform—ideal for rapid prototyping, user‑specific ergonomics, or brand‑driven product families.
